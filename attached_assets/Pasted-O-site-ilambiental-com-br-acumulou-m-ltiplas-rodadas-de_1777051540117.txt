O site ilambiental.com.br acumulou múltiplas rodadas de correções CSS mobile que estão em conflito entre si. O resultado é que a versão mobile continua com problemas de sobreposição do header fixo sobre o conteúdo. Você vai resolver isso de forma definitiva com uma única abordagem: adicionar um bloco CSS limpo e consolidado NO FINAL do arquivo de estilos principal, que sobrescreva todos os conflitos anteriores.
 
NÃO altere textos, cores, tipografia, estrutura, animações, hovers, formulário, links ou integrações. NÃO altere o desktop. NÃO remova regras antigas — apenas adicione o novo bloco ao final do arquivo CSS para garantir precedência.
 
---
 
DIAGNÓSTICO TÉCNICO CONFIRMADO:
 
Header fixo: .site-nav | position: fixed | top: 0 | z-index: 9999 | height: 73px
CSS variable: --nav-height: 73px
 
Problemas identificados no mobile (max-width: 767px):
1. .hero-animated-bg e .page-hero: padding-top oscila entre 88px e calc(var(--nav-height)+40px) por conflito de regras — resultado real: apenas 15px de respiro abaixo do nav. CRÍTICO.
2. .contact-hero-section: padding-top: 88px — mesmo problema. CRÍTICO.
3. .hero-content-inner: margin-left: 120px do desktop pode não estar sendo sobrescrito corretamente por conflito de especificidade. CRÍTICO.
4. .about-strip > div: width: 421px fixa do desktop vaza da tela. ALTO.
5. .contact-float-card: margin-top: -64px cria sobreposição no mobile. ALTO.
6. section h2, h1, p: font-sizes do desktop (44-64px) podem não estar sendo reduzidos consistentemente. MÉDIO.
 
---
 
SOLUÇÃO: Adicionar o bloco abaixo no FINAL do arquivo CSS principal (style.css, globals.css, main.css ou equivalente). Posicionado ao final, tem precedência sobre todas as regras anteriores conflitantes.
 
@media (max-width: 767px) {
