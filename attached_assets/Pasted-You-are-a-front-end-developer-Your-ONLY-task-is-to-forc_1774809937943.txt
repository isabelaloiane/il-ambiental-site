## STEP 7 — PATH VERIFICATION AND DEBUGGING

After all insertions, verify paths are correct:

### 7.1 — Check relative paths
If HTML files are in the root directory, the path should be: `src="assets/filename.ext"`
If HTML files are in a subfolder, the path should be: `src="../assets/filename.ext"`
If the project uses a /public folder, check if assets need to be in /public/assets

### 7.2 — Check for common path errors
Search all HTML files for these broken patterns and fix them:
- `src="/assets/` → might need `src="assets/` (no leading slash for static sites)
- `src="./assets/` → simplify to `src="assets/`
- `src="Assets/` → fix case sensitivity to match actual folder name
- `src="asset/` → fix to `src="assets/` (with 's')
- Double slashes: `src="assets//` → fix to `src="assets/`
- Spaces in path: encode or rename file

### 7.3 — Add a runtime check
Add this script to every HTML file, right before `</body>`:
```html
<script>
document.addEventListener('DOMContentLoaded', function() {
  var logo = document.querySelector('.site-logo-img');
  if (logo) {
    logo.addEventListener('error', function() {
      console.error('CRITICAL: Logo image failed to load from: ' + this.src);
      console.error('Attempting alternate paths...');
      var paths = [
        'assets/[EXACT_LOGO_PRINCIPAL_FILENAME]',
        './assets/[EXACT_LOGO_PRINCIPAL_FILENAME]',
        '/assets/[EXACT_LOGO_PRINCIPAL_FILENAME]',
        '../assets/[EXACT_LOGO_PRINCIPAL_FILENAME]'
      ];
      var current = this.src;
      for (var i = 0; i < paths.length; i++) {
        if (!current.endsWith(paths[i])) {
          this.src = paths[i];
          return;
        }
      }
    });
    if (logo.complete && logo.naturalHeight === 0) {
      logo.dispatchEvent(new Event('error'));
    }
  }
});
</script>
```

Replace `[EXACT_LOGO_PRINCIPAL_FILENAME]` with the real filename.
