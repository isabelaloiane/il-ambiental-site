Se o usuário está no final da página de contato e clica, a página não volta ao topo
Não há resposta visual de navegação perceptível ao usuário

2. Aplicar lógica de navegação inteligente
Substituir o comportamento atual por esta lógica:
javascript// Ao clicar no botão "Fale Conosco"
faleConoscoButton.addEventListener('click', (e) => {
  e.preventDefault();

  const contactPagePath = '/contato'; // ajustar conforme rota real do projeto
  const currentPath = window.location.pathname;

  // Verificar se já está na página de contato
  const isOnContactPage = currentPath === contactPagePath
    || currentPath === contactPagePath + '/'
    || currentPath.includes('contato')
    || currentPath.includes('contact');

  if (isOnContactPage) {
    // Já está na página de contato → scroll suave para o topo
    window.scrollTo({
      top: 0,
      behavior: 'smooth'
    });
  } else {
    // Está em outra página → navegar para contato
    // Definir flag para scroll ao topo após navegação
    sessionStorage.setItem('scrollToTop', 'true');
    window.location.href = contactPagePath;
  }
});
3. Garantir scroll ao topo ao chegar na página de contato
Na página de contato, adicionar lógica de inicialização:
javascript// Executar quando a página de contato carrega
document.addEventListener('DOMContentLoaded', () => {
  if (sessionStorage.getItem('scrollToTop') === 'true') {
    sessionStorage.removeItem('scrollToTop');
    window.scrollTo({ top: 0, behavior: 'instant' });
  }
});
4. Se o site usar framework com router (React Router, Next.js, etc.)
Adaptar a lógica para o sistema de rotas do projeto:
javascript// React Router / Next.js — exemplo de lógica
const handleFaleConosco = () => {
  if (location.pathname.includes('contato') || location.pathname.includes('contact')) {

      window.scrollTo({ top: 0, behavior: 'instant' });
    }, 100);
  }
};
